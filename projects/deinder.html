Restaurant D' Einder

2022 Angular NestJS TypeScript
MongoDB Bootstrap

Beschrijving

Dit was mijn eindproject voor mijn Software Developer opleiding. We moesten een website/applicatie maken voor een fictief restaurant gebaseerd op interviews van de medewerkers. Ik heb het gemaakt in een team van vier en de taal die we ervoor hadden gekozen was TypeScript.

We hebben de applicatie opgebouwd uit vier onderdelen waaraan ik allemaal heb gewerkt:

  • Een NestJS back-end met authenticatie
  • Een MongoDB instance in de cloud
  • Een Angular website voor klanten
  • Een Angular admin paneel voor medewerkers

De website heeft een groot aantal mogelijkheden. Bijvoorbeeld: Tafels reserveren als een klant, menu's en dranken kunnen toevoegen aan een winkelmandje, contact opnemen met het restaurant, inloggen en uitloggen, tafels, menu's en gerechten beheren als admin.
