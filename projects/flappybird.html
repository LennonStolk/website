Flappy Bird 3D

2024 C# Unity WebGL Universal Render Pipeline

Beschrijving

Flappy Bird 3D is een erg simpele game die ik in mijn eentje voor mijn opleiding heb gemaakt. Het doel in dit spel is om zo veel mogelijk diamantjes te verzamelen terwijl je middeleeuwse bouwwerken ontwijkt. Ik werkte aan deze game als game designer en programmeur.
Bij dit project heb ik het volgende ontworpen en geprogrammeerd:

  • Player controls
  • Dynamic obstacle loading system
  • Prefabs van gebouwen (met hitboxes)
  • Post processing
