F-Spot

2022 iOS/Android React Native Expo
TypeScript PHP MySQL

Beschrijving

Een muziek speler en playlist app die afgeleid is van Spotify. Ik heb de app gemaakt met React Native, en de back-end is gemaakt in PHP en MySQL. Ik heb PHP en MySQL gekozen omdat het op mijn goedkope webserver kan draaien.

Gebruikers kunnen inloggen, hun account bekijken, muziek toevoegen aan een playlist, de playlist wijzigen en hun muziek beluisteren.
