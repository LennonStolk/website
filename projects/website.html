Beschrijving

Dit is de website waar je nu opzit! Ik heb deze nieuwe website gemaakt omdat de oude qua design niet voldeed en toch nog niet af was. Ik heb 'm gemaakt met puur HTML en CSS om de website razendsnelle laadtijden te geven. En daardoor heeft de website ook goede SEO zodat die beter te vinden is op Google.

De website bevat ook een aantal verborgen REST API's die door middel van MySQL en PHP de backends vormen voor mijn andere applicaties, bijvoorbeeld die van F-Spot.
