World Eater

2024 Unity WebGL

Beschrijving

The world eater has devoured many planets before and is well known for it.
You must stop it. Use your attacks and jetpack to defeat it.
But don't wait to long.....

World Eater is een game die ik voor mijn opleiding heb gemaakt in een team van 4 man. Mijn rol in dit team was programmeur, game designer en scrum master. De opdracht was om een bossfight te maken met meerdere fasen en een unieke core mechanic. En wij hebben gekozen voor: Gravitational Pull.

Bij dit project heb ik het volgende ontworpen en geprogrammeerd:

  • Dialog system
  • Notification system
  • User Interface (main menu)
  • User Interface (game over)
  • Boss attack (shockwave)
  • Physics (jetpack)
  • Physics (dash)
  • Physics (bugfixing gravity)
