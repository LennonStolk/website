World Eater - Dialog System

The world eater has devoured many planets before and is well known for it.
You must stop it. Use your attacks and jetpack to defeat it.
But don't wait to long.....

World Eater is een game die ik voor mijn opleiding heb gemaakt in een team van 4 man. Ik werkte in dit team als scrum master, game designer en programmeur. De opdracht was om een bossfight te maken met meerdere fasen en een unieke core mechanic. Bij deze game is dit Gravitational Pull.

Bij dit project heb ik het volgende ontworpen en geprogrammeerd:

  • Dialog system
  • Notification system
  • Jetpack physics
  • Player dash action
  • Bugfixes voor de bestaande planeet physics
  • Polish voor de graphics
